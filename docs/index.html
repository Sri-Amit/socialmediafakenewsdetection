Privacy Policy – Social Media Fact Checker

Last updated: 2025-09-08

This Privacy Policy describes how Social Media Fact Checker handles information. When you click the extension action, the Extension may transmit a minimal text snippet from the visible page content to our API to compute credibility scores and surface citations.

Summary

  • Data collected (transmitted off-device): Website content (short text snippets you choose to check).
  • Not collected: DMs, credentials, payment info, or personal communications not visible on the page.
  • Optional technical data: IP address may appear in server logs for security and rate limiting; logs rotate regularly.
  • Sharing: We use service providers (“processors”) to host our API and perform AI inference (Google Gemini). Data is transmitted over HTTPS.
  • Selling/ads: We do not sell data, use it for advertising, or build browsing profiles.
  • Retention: Request snippets are processed ephemerally; operational logs (if any) are retained up to 30 days.

Information we collect and why

  • Website content (text snippets): Used to generate the credibility score and citations that you requested.
  • Technical data: Standard server logs (e.g., IP address, user agent) for security, abuse prevention, and debugging.

How we share information

We share data only with service providers acting on our behalf to provide the core functionality (hosting, AI inference). They are bound by agreements to process data solely for this purpose.

Local storage

We store minimal settings and a short-lived cache using chrome.storage.local. This data stays on your device.

Permissions

The Extension uses the minimum required permissions: activeTab, scripting, storage, and host access limited to selected sites and requested on demand.

Your choices

  • Disable or remove the Extension at any time.
  • Clear local cache and settings via the Extension’s options.
  • Contact us to request deletion of server logs associated with your IP, if retained.

Children’s privacy

The Extension is not directed to children under 13 and does not knowingly collect personal information.

Changes

We may update this policy. If changes are material, we will update the date above and provide a notice on the listing.
